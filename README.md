# Advanced Cryptography Assignment

This repository contains a collection of cryptography implementations including symmetric encryption, asymmetric encryption, and hashing utilities.

## Project Structure

- `main.py`: The main entry point for the application.
- `symmetric.py`: Implementation of a basic Caesar cipher.
- `asymmetric.py`: Implementation of modular exponentiation for asymmetric logic.
- `hashing.py`: SHA-256 hashing utility.
- `WEEK 1/`: 
    - Contains early work and initial encryption tests.

## Usage
Run `python main.py` to interact with the different cryptography modules.
